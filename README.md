# English:

## deCard
Allows you to customize Apple Pay cards. Only for [Jailbreak](https://www.cydiafree.com) users.

### Compiling
  - [Theos](https://theos.dev/) is required to compile the project.
  - You also may use Xcode.

### Compatibility
iPhone running iOS 13.x & 14.x & 15.x
